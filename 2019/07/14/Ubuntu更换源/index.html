Linux更换源

使用Ubuntu或Linux常出现源速率不够，导致下载或更新缓慢，原因是自带的源常为国外的源导致的。此时可以通过更换国内源来解决。

方法一：手动修改源文件进行源更换

* 备份源列表
* 修改源列表文件（注意区分系统代号）
* 更新源列表
* 更新软件包

备份源列表文件

源列表文件存放在 /etc/apt/sources.list

可直接copy sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak

修改源列表文件

同样目录下 sudo vi sources.list

若有图形化界面可使用gedit，更适合新手 sudo gedit sources.list

将内部源列表进行注释，把清华源复制进去，或阿里云源等

更新源列表

`sudo apt-get update`

更新源软件

`sudo apt-get upgrade`
